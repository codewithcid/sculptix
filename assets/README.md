# App assets

Drop your branding here and re-add the references in `app.json`:

| File                 | Size (px)   | Used for                         |
| -------------------- | ----------- | -------------------------------- |
| `icon.png`           | 1024×1024   | App icon (`expo.icon`)           |
| `adaptive-icon.png`  | 1024×1024   | Android adaptive foreground      |
| `splash.png`         | 1284×2778   | Splash screen (`expo.splash.image`) |
| `favicon.png`        | 48×48       | Web favicon                      |

The project currently runs with Expo's default icon/splash so you can start
developing immediately. Add these files and the matching keys in `app.json`
before shipping to the stores.
